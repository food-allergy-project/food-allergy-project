Landing Page


Content Strategy

Landing page will consist of the following contents. Starting from the top of the page:

Website Logo:

A logo that contains a symbol and the name of the website.

Navigation Bar:

Top navigation bar that included login option. Whenever the login option on the navigation bar is clicked, the page will automatically send the user into a login page.

About Page:

This section will be about Allergic-foodies. It will be a brief description about the website, and what the users can expect from taking the quiz.

Take Quiz:

A simple “Take Quiz” button will be centered on the mid way of the page. When the users clicked the “Take Quiz” button, they will be redirect onto a sign-up page. Below the button, will include an image of a food recipe.
